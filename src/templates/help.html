Create a normal campaign
Step 1: Select campaign type(Normal).
Step 2: Fill up the basic campagin details.
  • choose mode ( Open campaign or Fixed customers)
    1. If you wants to create open campaign for all customers you have to select mode as open customer.
    2. If you wants to create campaign for specific customers you have to select mode as fixed customer and upload a mobile csv file.
  • In sms template you have to add %cpn% as place holder for the coupon.
  • Select City.
    1. If you want to select all cities then check all cities else uncheck it and add particular cities from dropdown.
    2. Otherwise All city selected.
  • Select Branch.
    1. If you want to select all branches then check all branches else uncheck it and add particular branches from dropdown.
    2. Otherwise All branches selected.
  • click on next button.
Step 3: fill the configration detail for a campaign.
  • Enter the pax and voucher code inside the header settings.
  • click on next button.
Step 4: see your campagin details.
  • check your campaign details if details are correct then click on submit to finish else click on previous to edit.
Create a survey campaign
Step 1: Select campaign type(Survey).
Step 2: Fill up the basic campagin details.
  • choose mode ( Open campaign or Fixed customers)
    1. If you wants to create open campaign for all customers you have to select mode as open customer.
    2. If you wants to create campaign for specific customers you have to select mode as fixed customer and upload a mobile csv file.
  • In sms template you have to add %cpn% as place holder for the coupon.
  • Select City.
    1. If you want to select all cities then check all cities else uncheck it and add particular cities from dropdown.
    2. Otherwise All city selected.
  • Select Branch.
    1. If you want to select all branches then check all branches else uncheck it and add particular branches from dropdown.
    2. Otherwise All branches selected.
  • click on next button.
Step 3: Fill up the Questionnaire details.
  • select question type.(single select , multi select ,open text and fill text)
    1. If you wants to create the single or multi select question then you have to add answers for that question.
    2. If you wants to create the open text question select type as open text.
    3. If you wants to create fill in blank question you have to select type fill in blank and add %_% as place holder for answer.
  • click on next button.
Step 4: fill the configration detail for a campaign.
  • Enter the pax and voucher code inside the header settings.
  • click on next button.
Step 5: see your campagin details.
  • check your campaign details if details are correct then click on submit to finish else click on previous to edit.
